Tab 1

Capital Overview — Live fund metrics

The first screen after Fund Manager login. Updates immediately as stages and ticket sizes change.

What you see

  • 4 KPI cards — Target Raise, Total Committed, Total Funded, % to Close
  • Ticket size distribution chart (capital grouped by investment size bucket)
  • Stage funnel — investor count per pipeline stage

How to use it

  • Use the funnel to find bottlenecks (where investors stall)
  • Use the distribution to see if raise relies on whales vs broad base
  • Keep ticket sizes updated — everything aggregates from that number
Tab 2

Task Manager — All tasks in one list

The bell icon in the dashboard header shows a live count of open tasks.

What you see

  • Investor name, task description, type (⚡ Auto / Manual)
  • Priority (high / medium / low), due date, completion checkbox

Task types

  • ⚡ Auto — generated when investor moves stages (standard checklist for that stage)
  • Manual — created via +, link to investor, set due date / priority / notes
  • Filter by investor, stage, priority, or completion status
Tab 3

Investor Profiles — Fund investor database

List columns

  • Name, investor type, nationality, pipeline stage
  • Relationship strength badge, expected ticket size
  • Persona match % badge

Edit mode

Click Edit in the top-right corner of the profile panel → modify any field → Save to apply immediately.

Tab 4

Fundraising Pipeline — 13-stage Kanban board

What you can do

  • Drag & drop investor cards between stage columns — saves instantly
  • Auto-tasks for the new stage are created in Task Manager the moment a card is dropped
  • Scroll horizontally to navigate all 13 stages

Investor cards show

  • Name, expected ticket size, relationship strength badge
  • Stage column header shows investor count for that stage

Tab 5

Communication Center — Inbox · Call Logs · Templates

Inbox

  • Connect Gmail via OAuth (Settings → Gmail → Connect)
  • Investor emails auto-tagged by matched addresses
  • Compose and reply without leaving the portal

Call Logs

  • Log calls: investor, date, outcome (Interested / No Answer / Connected / Follow-Up Needed), notes
  • Toggle auto-create follow-up task linked to the investor

Templates

  • Reusable email templates with merge variables
  • {{investor_name}} {{fund_name}} {{ticket_size}}
  • Select a template when composing → auto-populates all fields
Tab 6

Research Capture — Review Chrome Extension captures

What you see

  • Capture card: source URL, capture date, extracted fields (name, firm, job title, LinkedIn, emails, phone)
  • Status: Pending / Accepted / Rejected

Accept flow

  1. Select a fund from the dropdown on the capture card
  2. Click Accept
  3. Investor is created in the fund's database immediately
  4. Dashboard switches to Investor Profiles and opens the new profile automatically
Tab 7

Personas — Define archetypes + score investors (0–100%)

What you can do

  • Create Persona — type, nationalities, sectors, age range, ticket size range
  • Run AI Analysis — scores every investor vs every persona (0–100%)
  • Analyze & Suggest — AI suggests archetypes not covered by existing personas

Where scores appear

  • Investor list: badge % on each investor card
  • Investor detail panel → Personas tab: progress bars per persona
  • Use 50%+ as the "match" threshold for coverage tracking
