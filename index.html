How to use the
Fund Manager Portal

This guide explains what each module does, what each role can see, and how the Chrome Extension feeds investor research into the portal.

Open Portal Login View GitHub Repo Extension Docs Style Ref
The portal is role-based: Admin has global visibility across all funds/users/investors. Fund Managers only see data for funds assigned to them (fully isolated).
Quick start

Start here (60 seconds)

  • Open /login and sign in with a demo role.
  • Admins: confirm Users/Funds/Assignments are configured.
  • Fund Managers: select a fund in the header, then operate within the 7 tabs.
  • Use the Chrome Extension to capture investors and review them in Research Capture.
