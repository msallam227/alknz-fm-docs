ALKNZ-Extension-v12

Chrome Extension
How it works

Captures investor profiles from the web and sends them to the portal for review in Research Capture (Fund Manager) and Investor Requests (Admin).

Open Portal Login Portal Repo Extension Docs (reference)
Folder: ALKNZ-Extension-v12/ • Chrome • Manifest V3
Purpose

One clean pipeline

  • Capture from LinkedIn and any webpage
  • Enrich with notes + relationship context
  • Upload to portal → review → accept into a specific fund
Install

Load unpacked extension

  1. Open chrome://extensions/
  2. Enable Developer Mode (top-right)
  3. Click Load unpacked → select ALKNZ-Extension-v12/
  4. Pin the ALKNZ icon to toolbar
  5. Right-click icon → Options → set Backend URL → log in with portal credentials
  6. Click Test Connection → should show “Connected ✓”
Capture Flow

From web → portal → fund

  1. Open a LinkedIn profile / company page (or any investor bio page)
  2. Click ALKNZ icon → side panel opens
  3. Extension auto-extracts: name, job title, firm, LinkedIn URL
  4. Add notes + relationship context + missing fields
  5. Click Upload to Portal
  6. Portal → Research Capture (FM) → capture appears as pending
  7. Select a fund → click Accept → investor created and profile opens automatically
