ALKNZ-Extension-v12 · Chrome · Manifest V3

Chrome Extension
How it works

Captures investor profiles from the web and sends them to the portal for review in Research Capture (Fund Manager) and Investor Requests (Admin).

Open Portal Login Portal Repo Extension Docs
Purpose

One clean pipeline

  • Capture from LinkedIn and any webpage
  • Enrich with notes and relationship context
  • Upload to portal → review → accept into a specific fund
Install

Load unpacked extension

  1. 1
    Open Extensions

    Navigate to chrome://extensions/ in Chrome.

  2. 2
    Enable Developer Mode

    Toggle Developer Mode on (top-right corner of the extensions page).

  3. 3
    Load Unpacked

    Click Load unpacked → select the ALKNZ-Extension-v12/ folder.

  4. 4
    Pin the icon

    Click the puzzle icon in Chrome's toolbar and pin the ALKNZ extension.

  5. 5
    Configure

    Right-click the ALKNZ icon → Options → set Backend URL → log in with your portal credentials.

  6. 6
    Test Connection

    Click Test Connection — should show "Connected ✓" with your investor count.

Capture Flow

From web → portal → fund

  1. 1
    Open a LinkedIn profile / company page (or any investor bio page)
  2. 2
    Click the ALKNZ icon to open the side panel
  3. 3
    Extension auto-extracts: name, job title, firm, LinkedIn URL — fields appear pre-filled
  4. 4
    Add notes + relationship context + any missing fields
  5. 5
    Click Upload to Portal — success confirmation appears in the panel
  6. 6
    Portal → Research Capture (FM) or Investor Requests (Admin) → capture appears as pending
  7. 7
    Select a fund → click Accept → investor is created and the profile opens automatically
